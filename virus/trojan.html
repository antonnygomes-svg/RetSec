O Cavalo de Troia (Trojan Horse) é um tipo de malware que se disfarça de software legítimo para enganar o usuário e infectar o sistema. Ao contrário dos vírus tradicionais, ele não se replica sozinho e exige que a própria vítima execute o arquivo malicioso. O termo técnico "vírus Cavalo de Troia" é amplamente utilizado no dia a dia, embora os especialistas classifiquem a ameaça estritamente como malware.

    Como Funciona a Infiltração: Engenharia social: Hackers usam faturas falsas ou alertas urgentes para induzir o clique. Anexos de e-mail: Arquivos executáveis vêm camuflados como documentos PDF ou imagens. Downloads piratas: Jogos modificados, cracks de ativação e mídias via torrent trazem o código oculto. Atualizações falsas: Janelas pop-up alertam sobre supostas falhas de segurança no navegador.

    Principais Tipos de Trojans: Backdoor: Abre uma porta dos fundos para o invasor controlar o computador remotamente. Trojan-Banker: Rouba dados bancários, senhas e informações de cartões de crédito. Trojan-Downloader: Instala secretamente outras ameaças adicionais, como ransomware. Trojan-Ransom: Modifica ou criptografa dados para exigir o pagamento de um resgate. Trojan-Spy: Monitora atividades na tela, registra teclas digitadas e acessa a webcam.
